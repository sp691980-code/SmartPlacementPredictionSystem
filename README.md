# 🎓 Smart Placement Prediction System

A Java Console Based Placement Prediction System that predicts a student's placement chances based on academic and technical performance.

---

## 📌 Features

- 🔐 Login System
- 👨‍🎓 Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 🏆 Top Performer
- 📊 Statistics Dashboard
- 📈 Sort Students
- 💾 Save & Load Student Data
- 📄 Export Report
- 🎯 Placement Score Prediction
- 🏢 Recommended Company
- 🎓 Grade System
- 🆔 Student ID
- 📅 Registration Date

---

## 💻 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- Collections (ArrayList)
- Exception Handling
- Git
- GitHub
